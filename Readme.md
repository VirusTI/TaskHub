# TaskHub

Консольный менеджер задач на .NET / C#. Создание, поиск, редактирование задач, статистика, сохранение в файл и фоновая проверка дедлайнов.

## Запуск

```bash
dotnet run
```

При первом запуске файл `tasks.txt` создаётся автоматически после первого сохранения.

## Возможности

- **Создание задач** - название, описание, приоритет (Low/Medium/High), дедлайн, статус (New/InProgress/Done)
- **Просмотр** - все задачи, выполненные, невыполненные, с высоким приоритетом
- **Редактирование** - изменение названия, описания, приоритета, статуса
- **Удаление** задачи по Id
- **Поиск** - по названию, статусу или приоритету
- **Статистика** - общее количество, выполненные, просроченные, разбивка по приоритетам
- **Сохранение и загрузка** из файла (асинхронно, через `async/await`)
- **Фоновая проверка дедлайнов** - отдельный `Task` раз в 10 секунд проверяет просроченные задачи и выводит уведомление

## Управление

Приложение работает через текстовое меню - выбор пункта вводится числом (`1`-`8`, `0` - выход). Задачи автоматически сохраняются при выходе.

## Структура

| Файл | Назначение |
|------|-----------|
| `Program.cs` | Точка входа, меню, обработка ввода |
| `Enums.cs` | Перечисления `Priority` и `Status` |
| `TaskItem.cs` | Класс задачи и абстрактный базовый класс |
| `TaskManager.cs` | Логика: CRUD, поиск, фильтры, статистика |
| `FileStorage.cs` | Асинхронное сохранение/загрузка |
| `DeadlineWatcher.cs` | Фоновая проверка дедлайнов |
